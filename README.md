# bluebell
帖子后端项目

## 项目介绍
- bluebell是一个仿reddit的社区帖子项目
- 基于gin框架和Vue框架搭建的前后端分离的web项目

## 项目技能

- gin框架
- viper配置管理
- JWT认证
- Zap日志库
- Go语言操作Mysql
- Go语言操作Redis
